Reproducibility notes

  • All analysis notebooks include raw-data provenance and calibration steps.
  • Hardware configurations are recorded with timing and phase-lock settings.
  • Signal processing pipelines maintain parameter logs for each run.

How to cite

Please cite the relevant publication and include dataset DOI if you reuse data or code. Suggested citation format:

Vicentini, E. et al., "Title of work", Year. Dataset DOI or software version.
